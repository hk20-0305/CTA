# Clinical Trial Eligibility Automation App

## Overview
A production-ready system for automating clinical trial eligibility screening in hospitals. The app uses AI/ML to match patient records with trial protocols, providing explainable decisions with confidence scores.

## Features
- Automated eligibility checks from PDFs
- Explainable AI decisions with confidence scoring
- Audit trail for regulatory compliance
- Dashboard for patient and trial management
- Flutter and web interfaces

## Technologies
- **Backend:** Python, FastAPI, BioBERT, PostgreSQL
- **Frontend:** Flutter (mobile), React (web)
- **NLP:** Hugging Face Transformers, sentence-transformers
- **Deployment:** Docker, Kubernetes
