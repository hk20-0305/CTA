# Clinical Trial Eligibility Web Dashboard

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Start the backend server (FastAPI)
4. Run `npm start`

## Features

- Dashboard with eligibility checks
- Patient and trial management
- Eligibility result display
- Responsive design

## Technologies

- React (Frontend)
- FastAPI (Backend)
- PostgreSQL (Database)
- BioBERT (NLP)
